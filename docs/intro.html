Introduction

Welcome to the official Grafunc documentation. In this documentation you can find the basics of how to use this program, features, limitations, and change log.

For licensing information, the source code, issues, or just follow development check out the GitHub Repository

This project is in its very early stages. Many of the planned features have yet to be implemented, but will be documented whenever they become available.

Offline Documentation

To view the documentation offline, check out the GitHub Repository and download the source code and extract the docs/ folder. Then open the top-level index.html file in a web browser.
